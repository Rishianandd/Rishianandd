> YOLOv8 · OpenCV · ESP32 · Python

Smart helmet integrating real-time YOLOv8-based detection of speed breakers and traffic signboards. Built end-to-end: annotation pipelines, model training, sensor fusion, and hazard alerts.

### 🚗 Vehicle Detection & Classification — DRDO CVRDE
> PyTorch · SSD · Faster R-CNN · ResNet-50 · FPN

SSD + R-CNN hybrid pipeline for military vehicle detection achieving **90%+ accuracy** across 4+ environmental conditions (fog, night, rain, glare). Cut inference latency **~40%** vs R-CNN-only baseline using transfer learning.
